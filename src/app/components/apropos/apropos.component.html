Application Web créée par Hatim Atartour et Othmane Mdarhri.

Dans le cadre de la recherche sur les Frameworks JS demandée par Madame Adadi

L'application utilise la dernière version d'Angular 9.0.7 .

Dans cette application, on démontre l'utilisation des composants, des services, du Routage, recupération des données depuis une API ainsi que les directives *ng d'ANGULAR.

On utilise aussi charts.js, ng2-charts et ng2-search-filter

L'application utilise l'API public de javieraviles, pour le fetch des données.
